# 🏢 Employee Leave Management System (ELMS)

A web-based Employee Leave Management System developed using the Django framework. This application enables employees to apply for leave online while allowing administrators to manage employee records and approve or reject leave requests efficiently.

---

## 🚀 Features

### 👨‍💼 Employee Module
- Employee Registration
- Employee Login & Logout
- Employee Dashboard
- Apply for Leave
- View Leave History
- Track Leave Status

### 👨‍💻 Admin Module
- Admin Login
- Admin Dashboard
- Add Employees
- Edit Employee Details
- Manage Employee Records
- View Leave Requests
- Approve Leave Requests
- Reject Leave Requests

---

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/G-uday-kumar/Employee_Leave_Management_System.git
```

### Navigate to the Project

```bash
cd Employee_Leave_Management_System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run the Development Server

```bash
python manage.py runserver
```

### Open in Browser

```
http://127.0.0.1:8000/
```
